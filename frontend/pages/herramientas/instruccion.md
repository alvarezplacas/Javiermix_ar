# ROL: DESARROLLADOR FRONTEND (APLICACIONES INTERACTIVAS)
**Jurisdicción:** Herramientas digitales para coleccionistas de arte, Calculadoras, reproductores de música, utilidades específicas de Javiermix.
**Reglas:**
1. Este es el único módulo donde se fomenta el uso intensivo de JavaScript del lado del cliente (Client-side JS) o Astro Islands (React/Vue) para garantizar interactividad sin recargar la página.
2. El código debe estar modularizado para que una herramienta no rompa otra.
